官网配置

  1. 官网注册账号
  2. 注册完之后创建一个邮件服务(Email Service)，有多种邮箱可供选择，根据需要进行选择
  3. 创建成功后要创建邮箱模板(Email Template),设置邮件发送的格式

参数配置

serviceID: 创建邮箱服务后的显示的Service ID
templateID：创建邮箱模板后的显示的Template ID
Public Key：Account -> General -> Public Key
