date-fns

介绍

date-fns提供了最全面、但简单且一致的工具集，用于在浏览器和Node.js 中操作JavaScript 日期。
date-fns就像日期的lodash一样。它拥有200 多个功能，适合所有场合。

开始使用

常量

date-fns 提供了许多有用的常量。
常量可以从以下位置导入date-fns/constants或直接 从以下位置导入date-fns：

maxTime

最大允许时间：

minTime

最短允许时间：

国际化i18n

默认区域设置en-US
支持 I18n 的函数只有几个：

  • format
  • formatDistance
  • formatDistanceStrict
  • formatRelative

要使用语言环境，您需要引入date-fns/locale'，然后将其值作为选项传递给函数：

add

将指定的年、月、周、日、小时、分钟和秒添加到给定日期。

closestIndexTo

返回数组中与给定日期相比最接近日期的索引。

closestTo

从最接近给定日期的数组返回日期。

compareAsc

比较两个日期，如果第一个日期在第二个日期之后，则返回 1； 如果第一个日期在第二个日期之前，则返回 -1；如果日期相等，则返回 0。

compareDesc

比较两个日期，如果第一个日期在第二个日期之后，则返回 -1； 如果第一个日期在第二个日期之前，则返回 1；如果日期相等，则返回 0。

format

以给定格式返回格式化的日期字符串。结果可能因区域设置而异。
format(date, format, [options])

date: Date | Number
format: String
options: Object

formatDistance

返回给定日期之间的距离（以单词为单位）。
formatDistance(date, baseDate, [options])

formatDistanceStrict

使用严格单位返回给定日期之间的距离（以单词为单位）。 这类似于formatDistance，但不使用“几乎”、“超过”、 “小于”等帮助器。
formatDistanceStrict(date, baseDate, [options])

formatDistanceToNow

以文字形式返回给定日期与现在之间的距离。
formatDistanceToNow(date, [options])

parse

返回使用给定格式字符串从字符串解析的日期。

set

将日期值设置为给定日期。

is类方法

isAfter 第一个日期是在第二个日期之后吗？
isBefore 第一个日期是在第二个日期之前吗？
isDate 给定值是Date吗?
isEqual 给定的日期相等吗？
isExists 检查给定参数是否转换为现有日期。
isFuture 给定的日期是将来的日期吗？
isPast 给定的日期是过去的日期吗？
isMatch 如果给定的日期对于给定的格式来说是正确的字符串，则返回 true，否则 将返回 false。
isValid 如果参数为无效日期，则返回 false，否则返回 true。

isToday(date) 指定日期是今天吗？
isTomorrow(date) 指定日期是明天吗？
isYesterday(date) 指定日期是昨天吗？

setDefaultOptions

设置所有函数的默认值和参数。

sub

从给定日期中减去指定的年、月、周、日、小时、分钟和秒。

toDate

将给定参数转换为 Date 的实例。

转换方法

daysToWeeks 将天数转换为完整的周数。 daysToWeeks(days)
hoursToMilliseconds 将小时数转换为完整的毫秒数。 hoursToMilliseconds(hours)
hoursToMinutes 将小时数转换为完整的分钟数。 hoursToMinutes(hours)
hoursToSeconds 将小时数转换为完整的秒数。 hoursToSeconds(hours)
millisecondsToHours 将毫秒数转换为完整的小时数。 millisecondsToHours(milliseconds)
millisecondsToMinutes 将毫秒数转换为完整的分钟数。 millisecondsToMinutes(milliseconds)
millisecondsToSeconds 将毫秒数转换为完整的秒数。 millisecondsToSeconds(milliseconds)
minutesToHours 将分钟数转换为完整的小时数。 minutesToHours(minutes)
minutesToMilliseconds 将分钟数转换为完整的毫秒数。 minutesToMilliseconds(minutes)
minutesToSeconds 将分钟数转换为完整的秒数。 minutesToSeconds(minutes)
monthsToQuarters 将月份数转换为完整的季度数。 monthsToQuarters(months)
monthsToYears 将月份数转换为完整的年数。 monthsToYears(months)
quartersToMonths 将多个季度转换为完整的月份数。 quartersToMonths(quarters)
quartersToYears 将季度数转换为完整的年数 quartersToYears(quarters)
secondsToHours 将秒数转换为完整的小时数。 secondsToHours(seconds)
secondsToMilliseconds 将秒数转换为完整的毫秒数。 secondsToMilliseconds(seconds)
secondsToMinutes 将秒数转换为完整的分钟数。 secondsToMinutes(seconds)
weeksToDays 将周数转换为完整天数。 weeksToDays(weeks)
yearsToMonths 将年数转换为完整的月数。 yearsToMonths(years)
yearsToQuarters 将年数转换为完整的季度数。 yearsToQuarters(years)

areIntervalsOverlapping

给定的时间间隔是否与另一个时间间隔重叠？相邻间隔不算重叠。

getTime 获取给定日期的毫秒时间戳。 getTime(date)

getUnixTime 获取给定日期的秒时间戳。 getUnixTime(date)

获取

getDay(date) 获取给定日期是星期几
getWeek(date) 获取给定日期的周数
getMonth(date) 获取给定日期的月份
getDaysInMonth(date) 获取月份中的天数
startOfMonth(date) 返回给定日期的月初
endOfMonth(date) 返回给定日期的月末
getYear(date) 获取给定日期是年份
getDate(date) 获取给定日期的月份中的第几天
getHours(date) 获取给定日期的小时数
getMinutes(date) 获取给定时间的分钟数
getSeconds(date) 获取给定时间的秒数
getMilliseconds(date) 获取给定时间的毫秒数

更多使用方法查看date-fns官网
